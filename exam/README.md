# 중간고사 응시 안내 (테스트용)

## 제출 파일
- **파일명**: `exam/answers.md` (정확히 이 경로)
- **인코딩**: UTF-8
- **브랜치**: `main`

## 답안 작성 형식 (엄수)

각 문항은 아래 형식. 형식 위반 시 해당 문항 **0점**.

### 단일선택
```
## Q1
답: 3
```
선택지 번호 숫자 1개.

### 복수선택 (모두 고르시오)
```
## Q3
답: 1, 3, 5
```
쉼표+공백 구분. 순서 무관.

### 참/거짓
```
## Q4
답: T
```
`T` 또는 `F` (대소문자 무관).

### 숫자
```
## Q5
답: 2.60
```
오차 허용 범위는 문항별 명시.

## 금지 사항
- Q 번호 누락, 중복, 범위 밖
- 답안 파일을 다른 경로에 두기
- 다른 브랜치에 push
